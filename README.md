# Материалы по Java (интенсив)

Репозиторий с сайтом-конспектом по темам. На GitHub лежит исходник; чтобы почитать материалы, откройте сайт у себя на компьютере — как это сделать, три варианта ниже.

Скачайте архив (зелёная кнопка **Code → Download ZIP**) или склонируйте репозиторий. Нужна одна папка со всеми файлами внутри.

---

## Как запустить

### 1. Самый простой путь — `Start.bat`

Понадобится [Node.js LTS](https://nodejs.org/) — ставится как обычная программа, можно всё оставить по умолчанию.

Зайдите в папку проекта (там, где лежит `Start.bat`), запустите **`Start.bat`** двойным щелчком. В первый раз подождите, пока подтянутся зависимости; потом обычно сам откроется браузер. Чёрное окно консоли не закрывайте, пока смотрите материалы. Остановить — **Ctrl+C** в этом окне.

### 2. Через терминал

Там же, в корне проекта:

```text
npm install
npm run dev
```

В браузере откройте адрес, который покажет терминал (часто `http://localhost:5173`). Терминал тоже не закрывайте, пока работаете.

### 3. Без Node — только файлы

В репозитории есть папка **`share-package`** — уже собранный сайт. Зайдите в неё и запустите **`Start-Lecture.bat`** или откройте **`index.html`**.

Не копируйте на стол один только `index.html` — рядом должны остаться папка **`assets`** и **`favicon.svg`**.
